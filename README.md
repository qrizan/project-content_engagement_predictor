## Content Engagement Predictor

Aplikasi prediksi engagement rate untuk konten TikTok & YouTube menggunakan Machine Learning (Random Forest Regressor).

## Notebook

https://colab.research.google.com/drive/1rltfHq8oBM_GbYDvmYGTYPEZP48UXcGH

## Jalankan Aplikasi

```
docker compose up --build
```
Akses aplikasi di: `http://localhost:5000`


## Tech Stack

- **Backend**: Flask + Python
- **Frontend**: Jinja2 + Bulma CSS
- **ML Model**: scikit-learn (Random Forest Regressor)
- **Deployment**: Docker + Railway
